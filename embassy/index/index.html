环境准备

已经迫不及待要开工了吗？稍等，要正式开始工作前，我们需要做一些准备。

在您的开发环境中，需要确保以下软件已正确安装：

  • rustup Rust工具链，参考官方链接或各发行版的安装文档
  • probe-run 一款固件写入工具，在将固件写入设备时需要。您也可以使用其他工具来做同样的事情，比如OpenOCD。

不用担心没有已支持的开发板，您可以在PC上，通过使用std来运行Embbassy应用。

支持的型号

Embassy支持很多型号的微控制器，不过最容易的方式就是拥有一块更常见的开发板套件。

nRF套件

  • nRF52 Dk
  • nRF9160 DK

STM32套件

  • STM32 Nucleo-144（STM32H743ZI MCU）开发板
  • STM32 Nucleo-144（STM32F429ZI MCU）开发板
  • STM32L4+ loT开发套件,支持低功耗无线, BLE, NFC, WiFi
  • STM32L0 开发套件，支持LoRa, Sigfox, 低功耗无线
  • STM32 Nculeo-64（STM32WL55JCI）开发版
  • STM32U5 loT开发套件

RP2040套件

  • 树莓派Pico

运行实例

首先从git仓库克隆代码到本地：

git clone https://github.com/embassy-rs/embassy.git
cd embassy
git submodule update --init

执行下列命令运行示例程序：

cd examples/nrf
cargo run --bin blinky --release
